# YouTube Watch Page — WEB101 PA1

A React/Next.js recreation of the YouTube Watch Page, built for the WEB101 Practical Assignment 1.

## Functionality
- Embedded video player using YouTube iframe
- Video title, views, like/dislike toggle, subscribe button
- Expandable video description
- Comment section with the ability to add new comments
- Recommendations sidebar with reusable video cards
- Fully responsive: desktop, tablet, and mobile layouts

## Tech Stack
- Next.js 
- Tailwind CSS
- JavaScript 

## Data Source
All data is sourced from `src/data/mockData.js`, which contains structured JavaScript objects for the video, comments, and recommendations.

## How to Run
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

## Tested On
- Chrome, Firefox, Edge
- Desktop (1280px+), Tablet (768px), Mobile (375px)
